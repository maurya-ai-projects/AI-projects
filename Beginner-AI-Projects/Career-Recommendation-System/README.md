## Description
Career Recommendation System is a beginner-level, rule-based AI project
that suggests suitable career options for students based on their
interest in subjects.

The system analyses user preferences and provides career guidance
using predefined decision rules.

---

## Objective
To understand how Artificial Intelligence systems can use rules and
logical conditions to make recommendations.

---

## Technologies Used
- Python

---

## AI Concept Used
Rule-Based Artificial Intelligence  
The program follows fixed rules to analyse user choices and recommend
careers.

---

## How It Works
1. The user answers questions about subject interests.
2. The system checks responses using rules.
3. Based on the answers, a suitable career option is suggested.

---

## Sample Output
Do you like Maths? yes  
Do you like Computers? yes  
Recommended Career: Computer Science / Artificial Intelligence
