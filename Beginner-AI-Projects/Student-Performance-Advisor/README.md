## Description
Student Performance Advisor is a beginner-level, rule-based AI project
that analyses a student's marks and provides academic feedback and
suggestions for improvement.

The program uses predefined rules to classify performance levels
and guide students on how to improve their studies.

---

## Objective
To demonstrate basic Artificial Intelligence concepts such as
decision-making and rule-based reasoning using Python.

---

## Technologies Used
- Python

---

## AI Concept Used
Rule-Based Artificial Intelligence  
The system follows fixed rules to analyse marks and give appropriate advice.

---

## How It Works
1. The user enters marks obtained (out of 100).
2. The program checks the marks using rules.
3. Based on the score, it gives performance feedback.
4. The student receives advice for improvement.

---

## Sample Output
Enter your marks: 72  
Performance Level: Good  
Suggestion: Focus on regular revision and practice.
